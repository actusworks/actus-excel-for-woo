== Frequently Asked Questions ==

= Can I export to CSV instead of Excel? =

The plugin uses Excel (.xlsx) format to support advanced features like dropdowns, validation, formulas, and helper columns. You can save as CSV in Excel if needed, but you'll lose the smart features.

= Will this work with variable products and variations? =

Yes! The plugin fully supports variable products and their variations, including attributes, default attributes, and variation-specific data.

= Can I transfer products between different WooCommerce sites? =

Absolutely! Export from Site A, import to Site B. The plugin uses SKU-first matching, perfect for cross-site transfers where IDs might differ.

= What happens if I import products with existing IDs? =

The plugin will update existing products with matching IDs or SKUs. No duplicates are created.

= Can I add new products via import? =

Yes! Leave the ID field empty for new products. The plugin will create them automatically.

= Is there a limit to how many products I can export/import? =

Yes, limits depend on your plan:
* **Free**: Up to 100 products
* **Professional**: Up to 5,000 products
* **Business**: Unlimited products

All plans handle batch processing efficiently to prevent timeouts.

= Can I edit the Excel file in Google Sheets? =

Yes! The Free version allows you to download Excel files and upload them to Google Sheets manually. However, **Pro and Business plans include direct Google Sheets integration** - export directly to Google Sheets and import back without downloading/uploading files. This is perfect for team collaboration!

= How many products can I manage? =

This depends on your plan:
* **Free**: Up to 100 products
* **Professional**: Up to 5,000 products
* **Business**: Unlimited products

= Can I manage multiple stores? =

Yes! You can connect:
* **Free**: 1 store
* **Professional**: 3 stores
* **Business**: 30 stores

= What's included in the Free version? =

The Free version includes:
* 1 store connection
* 1 custom template
* 1 custom query
* Up to 100 products
* Limited product properties
* Limited query filters
* Excel export/import with all smart features (dropdowns, validation, helper columns)

[Upgrade to Professional](https://awexcel.actusanima.com/#pricing) to unlock Google Sheets integration, more products, and unlimited templates!

= What if my import fails halfway through? =

The plugin processes imports in batches and reports which products succeeded/failed. You can fix issues and reimport failed products.

= Does this work with WooCommerce extensions? =

The plugin works with core WooCommerce fields. Custom fields from extensions can be exported as meta data, but may require manual handling.

= How do I improve validation errors? =

The validation system shows exactly what's wrong. Common fixes:
- Add SKUs to all products
- Ensure category/tag IDs exist
- Use correct data formats (true/false for booleans)
- Check that required fields (name, type) are present

= Does the plugin send my data anywhere? =

No! All processing happens on your server and in your browser. No data is sent to external servers. Your product data stays 100% private.

**Google Sheets Integration (Pro & Business)**: When you use the Google Sheets feature, data is transmitted securely via OAuth to your Google account only. You control access and permissions through Google's security system.

= Do I need a Google account for the plugin to work? =

No! The Free version works entirely with Excel files - no external accounts needed. Google account is only required if you want to use the Google Sheets integration feature (available in Pro & Business plans).
